# 🛡️ Multi-threaded Network Recon & OSINT Toolkit

A professional-grade Python security tool designed for rapid network reconnaissance and intelligence gathering. This project demonstrates core competencies in network programming, concurrency, and data persistence.

## 🚀 Key Features
- **High-Speed Scanning:** Utilizes Python's `threading` with a custom `Lock` mechanism to scan hundreds of ports concurrently without data corruption.
- **Service Fingerprinting:** Automatically identifies common services (SSH, FTP, HTTP, etc.) running on open ports.
- **OSINT Capabilities:** Integrates with Geolocation APIs to provide real-time intelligence including ISP, Country, and Coordinates of target IPs.
- **Automated Reporting:** Generates structured reports in both **TXT** and **JSON** formats for professional documentation.
- **Validation:** Includes robust IP address validation and error handling for real-world reliability.

## 🛠️ Technical Stack
- **Language:** Python 3.x
- **Libraries:** `socket`, `requests`, `threading`, `json`
- **Concepts:** Multithreading, Thread Safety (Locks), API Integration, Network Protocols.
